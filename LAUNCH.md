# Launch Kit

Distribution playbook for awesome-ai-cae. The repo is built to earn stars; this is how it gets discovered. **Lead with the finding, not the list:** *"I scored 100+ CAE tools on AI-readiness — only 2 are agent-native."*

## Pre-launch checklist

- [ ] **Social preview image** set in repo Settings → Social preview (use `media/banner.svg` exported to 1280×640 PNG, with the numbers strip).
- [ ] `README` above-the-fold reviewed on mobile width.
- [ ] `READINESS.md` + `data/readiness.json` fresh (weekly CI green).
- [ ] Pinned issue / Discussions enabled for "suggest a tool".

## Sequence (1 week)

| Day | Channel | Asset / hook |
|-----|---------|--------------|
| 1 | **trackawesomelist** submit + **awesome.re / sindresorhus/awesome** PR | repo already meets gates: 80+ days old, CC0, topics, awesome-lint ✅, badge ✅ |
| 1 | Cross-list PRs (see below) | one-line entry + the AI-Readiness angle |
| 2 | **Show HN** | `drafts/hackernews-showhn.md` — post Tue–Thu 8–10am ET |
| 3 | **r/CFD**, **r/MachineLearning**, **r/OpenFOAM** | refresh `drafts/reddit-*.md` to the Index hook first |
| 4 | **X / LinkedIn** (CAE + AI/agents) | screenshot of the Index table + "callability not stars" |

## Maintainer outreach (link-back loop)

Open a friendly issue on each listed MCP/agent project linking the Index ranking: LLNL/paraview_mcp, csml-rpi/Foam-Agent, Terry-cyx/MetaOpenFOAM, webworn/openfoam-mcp-server. Ask them to add a "ranked on awesome-ai-cae" line — builds the "referenced by" loop.

## Watch (KPI)

Stars/day, referrers (GitHub traffic), trackawesomelist rank, HN/Reddit position. Baseline before launch: external referrers ≈ 0 — the whole point is to move that number.
