> AI驱动的计算机辅助工程(CAE)工具精选列表。

## 这是什么

本列表收录了AI代理可以直接调用和自动化的CAE/CAD/CAM开源工具。专注于无需GUI即可执行仿真、可视化、设计、网格生成和优化的工具。

## 分类

- **MCP服务器** — AI原生工具接口
- **CFD** — 计算流体力学 (OpenFOAM, SU2, PyFR)
- **FEA** — 有限元分析 (FEniCS, CalculiX, MFEM)
- **SPH** — 光滑粒子流体动力学 (DualSPHysics, PySPH)
- **可视化** — 后处理与渲染 (VTK, ParaView, PyVista)
- **CAD与几何** — 参数化建模 (CadQuery, FreeCAD)
- **网格生成** — 网格生成器 (Gmsh, CGAL, NETGEN)
- **AI/ML** — 仿真加速 (Modulus, JAX-CFD, PhiFlow)
- **代理模型与PINNs** — 基于物理的神经网络
- **优化** — 设计优化 (OpenMDAO, pymoo)
- **数据格式** — I/O库 (meshio, h5py)
- **数据集** — 基准测试与训练数据
- **学习资源** — 教程与课程
