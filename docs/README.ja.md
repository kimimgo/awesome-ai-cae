> AI搭載のCAE（Computer-Aided Engineering）ツールのキュレーションリスト。

## これは何か

AIエージェントが直接呼び出して自動化できるCAE/CAD/CAMオープンソースツールを収集しています。GUIなしでシミュレーション、可視化、設計、メッシュ生成、最適化を実行できるツールに焦点を当てています。

## カテゴリ

- **MCPサーバー** — AIネイティブツールインターフェース
- **CFD** — 数値流体力学 (OpenFOAM, SU2, PyFR)
- **FEA** — 有限要素解析 (FEniCS, CalculiX, MFEM)
- **SPH** — 粒子法流体力学 (DualSPHysics, PySPH)
- **可視化** — ポスト処理・レンダリング (VTK, ParaView, PyVista)
- **CAD・ジオメトリ** — パラメトリックモデリング (CadQuery, FreeCAD)
- **メッシュ生成** — メッシュジェネレータ (Gmsh, CGAL, NETGEN)
- **AI/ML** — シミュレーション加速 (Modulus, JAX-CFD, PhiFlow)
- **サロゲートモデル・PINNs** — 物理ベースニューラルネットワーク
- **最適化** — 設計最適化 (OpenMDAO, pymoo)
- **データフォーマット** — I/Oライブラリ (meshio, h5py)
- **データセット** — ベンチマーク・学習データ
- **学習リソース** — チュートリアル・講義
